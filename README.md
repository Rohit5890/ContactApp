# ContactApp
Edit/Delete/Filter/Add Contacts

# Getting Started
Follow below commands in sequence to get started
```
git clone <repo url>

npm i 

npm start

```
Then open `http://localhost:9000/` to see working demo.

# Prerequisites
 **`node` setup is must prior to above steps**
